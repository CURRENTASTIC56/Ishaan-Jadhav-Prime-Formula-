# The Jadhav Sieve: Deterministic Algebraic Prime Sieve

This repository contains an optimized deterministic primality test. It utilizes an algebraic filter based on the sequence of primes taking the form of `6n ± 1`.

## The Core Mathematical Concept

It is a proven mathematical rule that all prime numbers greater than 3 take the form of either `6n + 1` or `6n - 1`. However, not all numbers generated by these forms are prime (e.g., perfect squares or products of distinct primes). 

This algorithm bypasses standard trial division by solving for the exact points where `6n ± 1` produces a composite number. 

If a composite number is formed by multiplying two distinct primes $p_1$ and $p_2$ (which can be represented as $6x \pm 1$ and $6y \pm 1$), we can expand the multiplication to establish absolute exclusion rules for $n$.

### The Exclusion Rules

**1. For numbers in the form of $6n + 1$:**
The formula will yield a prime number 100% of the time as long as $n$ does not equal:
$$n \neq 6xy + x + y$$
$$n \neq 6xy - x - y$$

**2. For numbers in the form of $6n - 1$:**
The formula will yield a prime number 100% of the time as long as $n$ does not equal:
$$n \neq 6xy - x + y$$

## How the Algorithm Works
To avoid testing every combination of $x$ and $y$, the algorithm rearranges the equations to solve for $y$. We only need to iterate $x$ up to the square root boundary:
$$x_{max} = \sqrt{\frac{n}{6}}$$

If we plug $x$ into our rearranged formulas and $y$ results in a perfect integer, we have immediately proven the number is composite and simultaneously calculated its prime factors.

**Rearranged for $6n + 1$ composites:**
$$y = \frac{n - x}{6x + 1} \quad \text{or} \quad y = \frac{n + x}{6x - 1}$$

**Rearranged for $6n - 1$ composites:**
$$y = \frac{n + x}{6x + 1} \quad \text{or} \quad y = \frac{n - x}{6x - 1}$$

## Usage
Simply run `prime_sieve.py` and pass any integer into the `is_prime_algebraic(number)` function to determine its primality deterministically without brute-force division.
